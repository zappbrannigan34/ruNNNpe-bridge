# Защита от убийства фоновых процессов

Настройки по модели телефона для стабильной фоновой работы.

## Samsung
- Настройки → Батарея → Фоновые ограничения
- RUNN Bridge → Без ограничений

## Xiaomi
- Настройки → Приложения → RUNN Bridge → Батарея → Нет ограничений
- Безопасность → Автозапуск → RUNN Bridge → ВКЛ

## Huawei
- Настройки → Батарея → Запуск приложений
- RUNN Bridge → Управлять вручную → всё ВКЛ

## Pixel / Stock Android
- Настройки → Приложения → RUNN Bridge
- Батарея → Неограниченное

## Советы по BLE-подключению
- NPE RUNN "засыпает" — периодический скан будит его
- Убедитесь что Bluetooth включён перед запуском
- RUNN виден только когда дорожка включена
- Один RUNN подключается к одному устройству за раз
- При проблемах: перезагрузите Bluetooth на телефоне
- CCC дескриптор обязателен для notification подписки
